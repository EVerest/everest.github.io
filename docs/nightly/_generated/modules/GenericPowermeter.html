10.1.11. GenericPowermeter¶

Powermeter driver for various powermeter hardware

For a detailed handwritten documentation see here

10.1.11.1. Provides¶

main:powermeter
Implementation of the driver functionality

model:

description:

description:

model:string
default: test_dummy
Selector for the powermeter configuration file to be used

default: 1
