auth_token_provider
###################

| Provide some authentication token


*********
Variables
*********
| **token**:*object*
|  Arbitrary auth token string and type of token provider (string provided in UI)
|  required: token, type
|  properties:
|   **token**:*string*
|    Arbitrary token string: this has to be printable case insensitive ascii: !!!FIXME!!! write a regex to allow only printable ascii
|   **type**:*string*
|    Type of token provider
|   **timeout**:*number*
|    Time (in s) the token can be considered valid by the auth framework
