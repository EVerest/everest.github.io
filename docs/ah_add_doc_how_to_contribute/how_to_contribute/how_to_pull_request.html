How To: Fork and Pull request¶

  1. Fork the repository you want to work on

    1. Go to the repository you want to fork.

    2. Click on the Fork button.

    3. Choose a name for your fork.

    4. Add a description for your fork.

    5. Click on the Create fork button.

  2. Create a branch in your fork

  3. Work on this branch

  4. Create a pull request to the original repository

    1. Click on the Pull request button.

    2. Click on the New pull request button.

    3. Click on “compare across forks”

    4. Choose the branch you want to pull.

    5. Choose the branch you want to push to, which should be the main branch of the original repository

    6. Click on the Create pull request button.

    7. Choose a title for your pull request.

    8. Choose a description for your pull request.

    9. Click on the Create pull request button or if you are still working on the pull request, select “Create draft pull request”.
