####################
auth_token_validator
####################

| Checks provided tokens for validity



********
Commands
********
| **validate_token**:*object*
|  Validate auth token and return result (with optional reason string)
|  **token**:*string*
|   Arbitrary token string: this has to be printable case insensitive ascii: !!!FIXME!!! write a regex to allow only printable ascii
|  **Result**:*object* (:ref:`auth_token_validator-Result <auth_token_validator-Result>`)
|   Result object containing validation result enum value (key: result) and optional reason string (key: reason)
|   **There is an extended definition for this object** :ref:`here <auth_token_validator-Result>`.
