6.24. sunspec_reader¶

This interface defines a generic Sunspec reader, which can be used to fetch values from Sunspec devices when an implementation is given.

6.24.1. Variables¶

measurement:object
Measured dataset
required: timestamp
properties:
timestamp:number
Timestamp of measurement
value:number
Measurement value
