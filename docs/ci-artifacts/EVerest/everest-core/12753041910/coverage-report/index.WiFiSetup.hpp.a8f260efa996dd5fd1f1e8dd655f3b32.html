13 * SSID encoding
14 * From Wikipedia:

16 * in a natural language, such as English"
17 *
18 * wpa-cli escapes SSID strings in scan results; character values 32..126 are
19 * not converted. The following conversions are applied:
20 * - \" double quote
21 * - \\ backslash
22 * - \e escape (\033)
23 * - \n newline

25 * - \t tab
26 * - \xnn for other values
