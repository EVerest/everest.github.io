.. JsYetiSimulator

******************************
JsYetiSimulator
******************************

Description
###########
SIL simulator for YETI hardware v1.0


Provides
########
| **board_support**:*board_support_AC*
| provides the board support Interface to low level control control pilot, relais, rcd, motor lock

| **debug_keepalive**:*debug_json*
| Provides the keepalive as a json object

| **debug_powermeter**:*debug_json*
| Provides the powermeter as a json object

| **debug_state**:*debug_json*
| Provides the state as a json object

| **debug_yeti**:*debug_json*
| provides the debug information of the charging driver

| **powermeter**:*powermeter*
| provides the Yeti Internal Power Meter

| **yeti_extras**:*yeti_extras*
| extra functionality special for Yeti

| **yeti_simulation_control**:*yeti_simulation_control*
| Interface for the Yeti HIL simulator
