3 /*
4 * Charger.h
5 *
6 * Created on: 08.03.2021
7 * Author: cornelius
8 *
9 * IEC 61851-1 compliant AC/DC high level charging logic
10 *
11 * This class provides:
12 * 1) Hi level state machine that is controlled by a) events from evse_board_support interface
13 * and b) by external commands from higher levels
14 *
15 * The state machine runs in its own (big) thread. After plugin,
16 * The charger waits in state WaitingForAuthentication forever. Send
17 * Authenticate()
18 * from hi level to start charging. After car is unplugged, it waits in
19 * ChargingFinished forever (or in an error state if an error happens during
20 * charging).
21 */
